🏢
사내 인트라넷 · 그룹웨어

E-그룹웨어

흩어진 근태, 결재, 인사, 자산, Help Desk 업무를 하나의 사내 포털로 통합합니다. 부서별 운영 방식에 맞춰 필요한 모듈부터 단계적으로 도입할 수 있습니다.

  • 근태관리 자동 집계 · CSV 내보내기
  • 업무일지 작성 · 관리자 검토 워크플로
  • 전자결재 워크플로 (기안→결재→완료)
  • 인사관리 · 급여 모듈 통합
  • 자산관리 · 유지보수 이력 · Help Desk
  • 조직도 · 게시판 · 사내 메일
All-in-One Intranet SQLite 경량 DB 도입 상담 기술 검토
💬
Messenger

E-Messenger

사내 대화와 파일 공유가 외부 메신저에 흩어지지 않도록 회사 전용 커뮤니케이션 채널을 제공합니다. PC, 웹, Android 환경을 함께 검토해 배포 방식을 정합니다.

  • 실시간 채팅 · 채널 관리
  • 파일 공유 · 읽음 확인
  • Windows 데스크톱 앱 (Electron)
  • Android 앱 (APK 직접 배포)
  • 관리자 패널 · 사용자 관리
Cross-Platform 도입 상담 화면 데모 기술 검토
🖥️
자산 · 티켓 · 라이선스 통합관리

E-자산통합관리

장비, 비품, 장애 접수, 라이선스 만료 이력이 엑셀과 메신저에 흩어지는 문제를 줄입니다. 자산 현황과 대응 이력을 한 화면에서 추적할 수 있게 구성합니다.

  • 장애 티켓 시스템 (접수/진행/완료)
  • 자산 · 유지보수 이력 관리
  • 소프트웨어 라이선스 만료 알림
  • PPT 보고서 자동 생성
  • 이메일 알림 · 마스터/서브 권한
ITSM Lightweight 도입 상담 기술 검토
🤖
사내용 AI챗봇 · API형

E-AI챗봇 (API형)

보고서, 메일, 공지, 업무 문서 초안을 빠르게 정리해야 하는 실무자를 위한 사내 AI 도구입니다. 클라우드 API 사용이 가능한 조직에 맞춰 빠른 응답과 편의 기능을 제공합니다.

  • 상시 부유창 위젯 (화면 우측 하단)
  • 실시간 스트리밍 응답
  • 문서·이메일·보고서 초안 작성
  • 이미지·텍스트 파일 첨부 지원
  • 캐릭터·프롬프트 커스터마이징
AI Powered · Groq 도입 상담 기술 검토
🔐
사내용 AI챗봇 · 온프레미스

E-AI챗봇 (온프레미스)

민감한 문서를 외부 서비스로 보내기 어려운 조직을 위한 내부망 AI 도입 옵션입니다. 인터넷 연결이 제한된 환경에서도 보안 정책에 맞춰 배포 범위를 검토합니다.

  • 완전 오프라인 동작 (Ollama + Gemma)
  • 데이터 외부 전송 0% — 내부망만 통신
  • USB 오프라인 배포 지원
  • 경량(7.2GB) / 고품질(9.6GB) 모델 선택
  • Essence와 동일한 UX · 커스터마이징
Secure · Offline AI 도입 상담 기술 검토
🛡️
ENDPOINT SECURITY

E-Security

PC별 보안 상태와 USB 사용 정책을 중앙에서 확인해야 하는 조직을 위한 엔드포인트 관리 콘솔입니다. Windows 업무 PC 기준으로 필요한 보안 범위를 상담 후 정리합니다.

  • 악성코드 스캔 (해시·패턴 기반)
  • USB 이동식 저장소 정책 제어
  • 중앙 관리 콘솔 (React 대시보드)
  • 위협 격리 및 감사 로그
Windows Agent 실시간 보호 보안 기능 상담
